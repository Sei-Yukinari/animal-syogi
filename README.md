# Animal Syogi

A Next.js implementation of Animal Shogi (どうぶつしょうぎ) - a simplified version of Shogi designed for children.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
